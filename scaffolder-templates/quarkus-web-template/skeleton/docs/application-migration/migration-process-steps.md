# 마이그레이션 프로세스 단계

1. 사전 준비: 개발자를 위한 온보딩
2. 애플리케이션 현대화 - 코드를 컨테이너에 맞게 조정
   1. 검증된 방법론을 적용하여 소스 코드를 컨테이너로 이전할 준비를 완료합니다.
3. 컨테이너 이미지 빌드
4. 컨테이너 이미지 실행 (배포)
5. 통합:
   1. 가이드라인과 모범 사례를 이해합니다.
   2. 애플리케이션을 다른 시스템(보안, 로깅)과 통합합니다.
6. 애플리케이션을 LLE로 승격:
   1. 애플리케이션을 이동하기 위한 가이드라인과 절차를 이해합니다.
   2. 서로 다른 환경에서 변경되는 사항을 다시 점검합니다.
      1. 필요한 시크릿 생성
      2. 필요한 ConfigMap 생성
7. 프로덕션 가이드라인 및 TODO 목록:
   1. Red Hat 프로덕션 가이드라인 검토
   2. 프로덕션 가이드라인 검토
   3. 프로덕션 TODO 목록 업데이트
